## Use this extension

此仓库可以作为 **插件** 添加到 MakeCode 中。

* open https://arcade.makecode.com/
* 点击 **新项目**
* 点击齿轮图标菜单下的 **扩展**
* search for the URL of this repository and import

## Edit this extension

在 MakeCode 中编辑此仓库。

* open https://arcade.makecode.com/
* 点击 **导入**，然后点击 **导入 URL**
* paste the repository URL and click import
